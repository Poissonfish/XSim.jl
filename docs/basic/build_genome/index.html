Quick start

build_genome(n_chr    = 2,
             n_marker = 10000)

By a file

build_genome("map.csv")

By a dataframe

using DataFrames
data = CSV.read("map.csv", DataFrame)
build_genome(data)

Load with pre-load reference

build_genome("map.csv"; species="cattle")
build_genome(data;      species="cattle")
